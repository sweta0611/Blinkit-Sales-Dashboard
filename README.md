# Blinkit-Sales-Dashboard | Power BI Project
An interactive Power BI dashboard built to analyze Blinkit’s sales performance, highlighting key metrics, outlet distribution, and product-wise insights for better business decision-making.

## Overview

This project presents an interactive sales dashboard for Blinkit – India’s last-minute delivery app.
The dashboard is built using Power BI to analyze sales performance, customer preferences, and outlet insights.

## Objective

To analyze and visualize:
* Sales performance across different outlet locations and sizes
* Product category contribution to total sales
* Customer preferences based on fat content and item types
* Trends in outlet establishment over time

## Key Metrics

*  **Total Sales:** $1.20M
*  **Number of Items:** 8523
*  **Average Rating:** 4.0
*  **Average Sales:** $141

## Key Insights

*  Tier 3 outlets generate the highest revenue contribution
*  Medium-sized outlets dominate overall sales
*  Top-performing categories include Fruits, Snacks, and Household items
*  Sales show steady growth with a peak around 2018–2019
*  Regular fat products contribute more compared to low-fat products

## Dashboard Features

* Interactive filters (Outlet Location, Size, Item Type)
* Sales breakdown by item category
* Outlet performance comparison
* Fat content analysis
* Time-based sales trend visualization

## Tools & Technologies

* **Power BI** – Dashboard creation & visualization
